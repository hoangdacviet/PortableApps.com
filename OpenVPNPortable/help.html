OpenVPN Portable-Specific Issues

  • Installing OpenVPN Portable
    • Download and run installer.

  • Using OpenVPN Portable
    • Log is written to Data\log.
    • Put .ovpn config files in Data\config.
    • Run OpenVPNPortable.exe as administrator (required for TAP driver installation).

  • Upgrading OpenVPN Portable
    • OpenVPN
      1. Download latest 32-bit OpenVPN installer from the official site.
      2. Extract installer using 7-Zip.
      3. Replace App\OpenVPN\bin with [Installer]\bin
    • TAP-windows
      1. Download latest TAP-windows source zip from the official site.
      2. Replace tap0901.cat, tap0901.sys and OemWin2k.inf for each architecture in App\OpenVPN\driver with the ones in the zip archive.

  • Modifications
    • TAP driver is installed before and removed after execution.
    • Registry entries are modified to load configs from and write logs to portable Data folder.

  • Version History
